Infinity ========

Infinity is a number that seems sort of mystical.

It is the biggest number. There isn't any number bigger than it, by definition. But is it really a number?

Let's try assuming infinity is a number and let's also try some operations on it.

First off, the most basic of operations. Adding. If I have 2 ducks, and you have 3 ducks, and we join them together, we now have 5 ducks. We can count those ducks one by one. That's why 2 + 3 = 5.

But the idea starts to break apart once we put in infinity. If I have inf ducks, and you still have 3 ducks, we can still join them together and start counting: 1, 2, 3, 4, 5, ..., inf. There's still infinity ducks, adding 3 ducks does nothing. So we can say inf + 3 = inf.

We could just run with it and go on, but you can see how this already creates problems. Here's an example:

``` Let's start with this equation. inf + 2 = inf

Then, we add 3 to both sides. inf + 2 + 3 = inf + 3 inf + 5 = inf + 3

We can now subtract inf from both sides. 5 = 3 ```

What just happened there? 5 is obviously not equal to 3. So something is obviously wrong. Either all of math is wrong and 5 is equal to 3, or our assumption is wrong.

It isn't looking good for our assumption, so that means infinity is not really a number. It's more of an idea, a concept. You can say something is "infinite", as in "there are infinite numbers", or maybe "the universe is (maybe) infinite", but you can't just plug infinity into formulas.

Limits ======

This is why some really smart people invented what's known as "limits".

A limit basically says "hey, I know infinity isn't really a number, but what happens if I get really really close?"

For example, euler's number, commonly known as e. Its definition is the following:

``` e = lim n -> inf (1 + 1/n)^n ```

Now let's try just naively plugging in infinity as n.

``` e = (1 + 1/inf)^inf

We can think of 1/inf as being zero, since dividing something to infinite people would cause each person to get basically nothing. e = (1 + 0)^inf e = 1^inf

Now what might 1^inf be? We know that 1 to the power of anything is 1, so let's just say it's 1. e = 1 ```

As some of you may know, e is not 1. It's about 2.718. So how did those smart people get that value?

They did so by not plugging in infinity itself, but by plugging in bigger and bigger values of n. Here's a table that shows what happens as you make n bigger and bigger.

|n|e| |-----| |1|2.0| |12|2.613035290224676| |52|2.692596954437168| |365|2.7145674820219727| |100000|2.7182682371922975|

You can see that slowly and slowly as we build up bigger and bigger values of n it's getting nearer to some specific value.

Maths people call this "approaching", basically getting nearer and nearer a value, but never touching it, because actually touching that value would require getting to infinity, which as we know is impossible if we just plug in larger and larger values of n. We'll never run out of numbers and say "okay now this is infinity".

You can see how this demostrates that limits aren't just plugging in infinity into our formula, or just one really big value, but increasing and increasing values until we can say "okay, if we keep going on this is what we'll get."

To really understand this, you can try it with other limits. The notion of "getting closer" is still there, even if what's after the arrow isn't infinity. If we have `n -> 1`, we would plug in values such as 0.1, then 0.2, then 0.5, 0.9, 0.999999999, etc... but never exactly 1.

Here's a fun limit I like:

``` lim n -> 1 (((n*n) - 1) / (n - 1)) ```
